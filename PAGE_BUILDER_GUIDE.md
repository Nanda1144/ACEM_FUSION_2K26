## ðŸ“‹ Page Builder Guide

### Accessing the Page Builder
1. Open Admin Dashboard (chatbot â†’ enter passkey)
2. Click **"Page Builder"** tab (first tab)
3. Select a page from the dropdown

### Creating Sections

#### What are Sections?
Sections are containers that hold components. Each page can have multiple sections:
- **Header Section**: Top of the page
- **Body Section**: Main content area
- **Footer Section**: Bottom of the page

#### How to Create a Section
1. In the Page Builder, look at the left sidebar
2. Scroll down to "Create Section"
3. Click one of:
   - **+ Header Section**
   - **+ Body Section**
   - **+ Footer Section**
4. Section appears in the canvas

### Adding Components

#### Available Components
1. **Text Box** - Regular paragraph text
2. **Heading** - Large heading text (H1-H6)
3. **Image** - Upload and display images
4. **Button** - Clickable buttons with links
5. **Card** - Content cards with title and description
6. **Spacer** - Add vertical spacing
7. **Divider** - Horizontal line separator

#### How to Add Components
1. Click any component from the left sidebar
2. Component is added to the first section
3. Click **"Edit"** to customize properties
4. Click **"Delete"** to remove

### Editing Components

#### Edit Mode
1. Hover over any component
2. Click **"Edit"** button (appears on hover)
3. Dialog opens with all properties
4. Modify values:
   - **Text content**
   - **Colors**
   - **Sizes**
   - **Alignment**
   - **Links**
5. Click **"Save Changes"**

#### Example: Editing a Text Box
```
Content: "Welcome to our college fest!"
Font Size: "lg" (options: xs, sm, base, lg, xl, 2xl)
Font Weight: "bold" (options: normal, bold)
Color: "#00D9FF" (use color picker)
Align: "center" (options: left, center, right)
```

#### Example: Editing a Heading
```
Content: "About Our Event"
Level: "h2" (options: h1, h2, h3, h4, h5, h6)
Font Size: "3xl"
Font Weight: "bold"
Color: "#FF00E5"
Align: "center"
```

#### Example: Editing an Image
```
Source URL: "https://your-image-url.com/image.jpg"
Alt Text: "Event Photo"
Width: "100%" (or "500px")
Height: "auto" (or "300px")
Object Fit: "cover" (options: cover, contain, fill)
```

#### Example: Editing a Button
```
Text: "Register Now"
Link: "https://forms.google.com/your-form"
Variant: "primary" (options: primary, secondary, outline)
Size: "lg" (options: sm, md, lg)
```

### Preview Mode

#### Switching to Preview
1. Click **"Preview Mode"** button (top right)
2. See exactly how visitors will see the page
3. Edit controls are hidden
4. Click **"Edit Mode"** to return

### Managing Multiple Pages

#### Selecting Pages
1. Use the dropdown at the top of the canvas
2. Select any page to edit
3. Each page has its own sections and components

#### Creating New Pages
1. Go to **"Pages"** tab
2. Click **"Add Page"**
3. Fill in details
4. Return to **"Page Builder"** tab
5. Select your new page from dropdown

## ðŸ’¡ Pro Tips

### Best Practices

1. **Plan Your Layout**
   - Sketch on paper first
   - Decide section structure
   - List needed components

2. **Use Sections Wisely**
   - One section per major area
   - Group related components
   - Keep it organized

3. **Component Order**
   - Components stack top to bottom
   - Add spacers for breathing room
   - Use dividers to separate topics

4. **Color Consistency**
   - Use same colors throughout
   - Match header/footer colors
   - Test contrast for readability

5. **Preview Often**
   - Check preview mode regularly
   - Test on mobile (resize browser)
   - Verify all links work

### Common Patterns

#### Hero Section
```
1. Body Section
2. Heading (h1, large, centered)
3. Text Box (description, centered)
4. Spacer (40px)
5. Button (call-to-action, centered)
```

#### Content Section
```
1. Body Section
2. Heading (h2, left-aligned)
3. Divider
4. Text Box (paragraph)
5. Image (full width)
6. Spacer (60px)
```

#### Call-to-Action
```
1. Body Section
2. Spacer (40px)
3. Heading (h3, centered)
4. Text Box (short text, centered)
5. Button (primary, centered)
6. Spacer (40px)
```

## ðŸ†˜ Troubleshooting

### Components Not Showing?
- Make sure section exists
- Check if component is in correct section
- Verify "is_visible" is enabled
- Refresh the page

### Can't Edit Component?
- Ensure you're in Edit Mode (not Preview)
- Click "Edit" button on hover
- Check if dialog opens
- Try refreshing page

### Footer Not Updating?
- Click "Save Changes" button
- Wait 2-3 seconds
- Refresh the website
- Check browser cache

### Colors Not Applying?
- Use hex format (#00D9FF)
- Click color picker for accuracy
- Save changes after editing
- Clear browser cache

### Images Not Loading?
- Check image URL is valid
- Ensure image is publicly accessible
- Try uploading to gallery first
- Use that URL in component
