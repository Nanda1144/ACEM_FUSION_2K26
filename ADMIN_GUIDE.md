## ğŸ’¡ Pro Tips

### Best Practices
- **Save Often**: Click "Save All Changes" after each modification
- **Preview**: Open website in new tab to see changes
- **Image Size**: Keep images under 1MB for faster loading
- **Consistent Colors**: Use the same color scheme throughout
- **Test Mobile**: Check how it looks on phone after changes

### Recommended Image Sizes
- **Logos**: 200x200px (square)
- **Event Images**: 800x600px (landscape)
- **Committee Photos**: 400x400px (square)
- **Gallery Images**: 1200x800px (landscape)
- **Background Images**: 1920x1080px (full HD)

### Color Combinations
- **Professional**: Navy Blue (#1E3A8A) + White (#FFFFFF)
- **Modern**: Cyan (#00D9FF) + Magenta (#FF00E5)
- **Classic**: Black (#000000) + Gold (#FFD700)
- **Fresh**: Green (#10B981) + White (#FFFFFF)

### Font Pairing
- **Modern**: Inter (headers) + Inter (body)
- **Classic**: Georgia (headers) + Arial (body)
- **Professional**: Verdana (headers) + Verdana (body)

## ğŸ†˜ Troubleshooting

### Changes Not Showing?
1. Make sure you clicked **"Save All Changes"**
2. Refresh the page (Ctrl+R or Cmd+R)
3. Clear browser cache (Ctrl+Shift+Delete)

### Can't Upload Image?
1. Check file size (must be <5MB)
2. Check format (JPG, PNG, GIF, WEBP only)
3. Try a different image

### Forgot Passkey?
- Contact the technical administrator
- Default passkey: `acemadmin@fusion`

### Logo Not Showing?
1. Make sure image uploaded successfully
2. Check if "Save All Changes" was clicked
3. Refresh the page
