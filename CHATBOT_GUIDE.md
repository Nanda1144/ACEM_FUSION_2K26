## üéØ How to Use

### For Visitors

#### Step 1: Open Chatbot
- Click the **blue circular button** with chat icon (üí¨) in the bottom-right corner
- The chatbot window will open

#### Step 2: Choose an Option
You'll see a welcome message with buttons:
- **About Fusion26** - Learn about the fest
- **View Events** - See all events
- **Committee Info** - Meet the team
- **Gallery** - View photos
- **Admin Access** - For administrators only

#### Step 3: Navigate Through Information
- Click any button to get information
- The chatbot will respond with details
- More buttons will appear for related topics
- Click "Back to Menu" to return to main options

#### Step 4: Ask Questions
- Type your question in the input field
- The chatbot understands natural language
- Examples:
  - "How do I register?"
  - "Tell me about technical events"
  - "Who are the coordinators?"
  - "Show me the gallery"

### For Administrators

#### Step 1: Open Chatbot
- Click the chatbot icon (bottom-right)

#### Step 2: Select Admin Access
- Click the **"Admin Access"** button
- Or type "admin" or "login"

#### Step 3: Enter Passkey
- The chatbot will ask for your passkey
- Type: **`acemadmin@fusion`**
- Press Enter or click Send

#### Step 4: Access Dashboard
- If passkey is correct: Success message appears
- Admin dashboard opens automatically
- If passkey is wrong: Error message with retry option

## üêõ Troubleshooting

### Chatbot Not Opening
- **Refresh page**: Press Ctrl+R or Cmd+R
- **Check icon**: Look for blue button in bottom-right
- **Clear cache**: Ctrl+Shift+R or Cmd+Shift+R
- **Check console**: F12 ‚Üí Console for errors

### Messages Not Appearing
- **Wait for typing**: Bot shows typing indicator
- **Check connection**: Ensure internet is working
- **Scroll down**: Messages might be below visible area
- **Restart chat**: Close and reopen chatbot

### Admin Access Issues
- **Check passkey**: Must be exactly `acemadmin@fusion`
- **No extra spaces**: Copy-paste carefully
- **Backend running**: Ensure server is active
- **Check network**: F12 ‚Üí Network tab for API calls

### Buttons Not Working
- **Wait for load**: Let page fully load
- **Check clicks**: Ensure you're clicking the button
- **Try typing**: Use text input as alternative
- **Refresh page**: Reload if buttons unresponsive
