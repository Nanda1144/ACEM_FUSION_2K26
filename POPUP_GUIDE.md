# Welcome Popup Feature - User Guide

## Overview
A welcome/announcement popup feature has been added to your Fusion26 website. This popup displays an image when users visit your site, perfect for announcements, promotions, or important information.

## How to Change the Popup Image

### Step 1: Access Admin Dashboard
1. Click the chatbot icon at the bottom-right corner of your website
2. Enter your admin passkey (default: `acemadmin@fusion`)
3. The Admin Dashboard will open

### Step 2: Navigate to Popup Management
1. In the Admin Dashboard, click on the **"Popup"** tab
2. You'll see the Popup Management interface

### Step 3: Upload Your Popup Image
1. **Option A - Upload Image:**
   - Click the "Upload" button
   - Select an image from your computer (max 1MB)
   - The image will be automatically uploaded and the URL will be filled in

2. **Option B - Enter Image URL:**
   - Paste an image URL directly into the "Popup Image" field

### Step 4: Configure Popup Settings

**Enable/Disable Popup:**
- Toggle the "Enable Popup" switch to show or hide the popup

**Link URL (Optional):**
- Add a URL if you want the popup to be clickable
- When users click the popup, it will open this URL in a new tab

**Display Settings:**
- **Show Once Per Session:** When enabled, the popup only shows once per browser session
- **Display Delay:** Set how many milliseconds to wait before showing the popup (1000ms = 1 second)

### Step 5: Preview and Save
1. Click "Full Preview" to see how your popup will look
2. Click "Save Settings" to apply your changes
3. The popup will now appear on your website according to your settings

## Recommended Image Specifications
- **Size:** 1200x800px or 1920x1080px
- **Format:** JPG, PNG, or WebP
- **File Size:** Maximum 1MB
- **Aspect Ratio:** 16:9 or 3:2 for best results

## Tips
- Use high-quality images with clear text for better visibility
- Keep important information in the center of the image
- Test the popup on both desktop and mobile devices
- Use the delay setting to avoid annoying users (recommended: 1-2 seconds)

## Testing Your Popup
1. Save your settings in the admin panel
2. Close the admin dashboard
3. Refresh your website
4. The popup should appear after the configured delay
5. To see it again, clear your browser's session storage or use an incognito window

## Troubleshooting
- **Popup not showing:** Make sure "Enable Popup" is turned ON and an image URL is provided
- **Image not loading:** Verify the image URL is correct and accessible
- **Popup shows every time:** Disable "Show Once Per Session" if you want it to appear on every page load
